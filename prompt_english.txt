Build a premium, high-fidelity landing page and interactive booking application for a bespoke software studio named "SOFTWARE ÚNICO". The application must showcase a modern, sleek glassmorphism aesthetic with a vibrant corporate purple (#6233f3) accent matching the brand identity.

Ensure all copy, navigation links, modals, forms, and validation alerts are written in native, premium English.

### General Design Guidelines
- **Color Palette**: Deep corporate navy (#1E325A), slate gray (#5E6470), background off-white (#f0f0f0), and vibrant corporate purple (#6233f3).
- **Typography**: Import and use "Plus Jakarta Sans" from Google Fonts for all branding signatures and elegant headers. For the branding label, use: `font-plus-jakarta uppercase tracking-[0.18em] font-light text-[#1E325A]` for "SOFTWARE" and `font-semibold text-[#6233f3]` for "ÚNICO".
- **Viewport Layout**: Compact gaps, tight margins, and optimized column widths to deliver a natively spacious, comfortable 75%-80% scaled aesthetic on standard 100% zoom screens.

---

### Dependencies
- Use `lucide-react` for high-quality SVG icons.
- Use `motion` (imported from `'motion/react'`) for clean, smooth hardware-accelerated animations.

---

### File Structure & Setup

#### 1. Global Styles (`src/index.css`)
Import the Google Font and configure the Tailwind CSS theme variables:
```css
@import url('https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@200;300;400;500;600;700&display=swap');
@import "tailwindcss";

@theme {
  --font-plus-jakarta: "Plus Jakarta Sans", ui-sans-serif, system-ui, sans-serif;
  --font-helvetica: "Helvetica Regular", ui-sans-serif, system-ui, sans-serif;
}

html {
  scroll-behavior: smooth;
}

body {
  margin: 0;
  overflow-x: hidden;
  background-color: #f0f0f0;
  font-family: var(--font-helvetica);
}
```

#### 2. App Controller (`src/App.tsx`)
Implement a client-side router managing transition states between two main views: `'home'` and `'booking'`. Use `<AnimatePresence mode="wait">` to animate navigation switches cleanly. Set global selection colors in the main container: `selection:bg-[#6233f3]/20 selection:text-[#1E325A]`.

#### 3. Navbar Component (`src/components/Navbar.tsx`)
A spacious, balanced header that scales gracefully:

- **Center Menu**: `hidden md:flex gap-5 lg:gap-8`. Include links: `Solutions`, `Services`, `Process`, `Technologies`, `Contact`.
  - **Premium Hover Chevron Animation**: Do not display chevron arrows permanently. Instead, wrap the chevron icon (`ChevronRight`, `w-3.5 h-3.5 text-[#6233f3]`) in a container that is completely hidden when idle (`w-0 opacity-0`) and smoothly slides/fades in ONLY on list hover (`group-hover:w-3.5 group-hover:opacity-100 transition-all duration-300 ml-0.5`).
- **Right Button**: `flex-initial shrink-0 flex justify-end`. Renders the rounded button "Schedule a Consultation" leading to the booking page, featuring an `ArrowUpRight` icon.

#### 9. Booking Page (`src/components/Booking.tsx`)
A full-screen, focused form container styled like a floating glass panel on a clean gray canvas.
- **Branding Header**: A simplified "Book a Consultation" section with the subtitle `"Enter your details and our engineering team will get in touch today to design your custom software solution."`
- **Fields**:
  - `Full name *` (Required)
  - `Company / Organization` (Optional)
  - `Phone or WhatsApp *` (Required)
- **Submit Action**: Connects to the EmailJS REST API (`https://api.emailjs.com/api/v1.0/email/send`) with fully simulated loading, error, and success states:
  - On submit, show a spinner and `"Processing..."` copy.
